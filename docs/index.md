# 🌅 Sunrise Design System

欢迎来到曦望设计系统！这是一套完整的设计 Token 文档，包含颜色、字体、间距、阴影等全面的设计规范。

## 🎯 快速开始

选择你感兴趣的部分：

- **[颜色 Token](/tokens/colors)** - 品牌色、深浅色模式、状态色
- **[字体 Token](/tokens/typography)** - 字体族、字号、字重、行高
- **[间距 Token](/tokens/spacing)** - 完整的间距系统
- **[其他 Token](/tokens/others)** - 阴影、图标、布局、动效

## 📋 文档信息

- **版本**: v1.0
- **更新日期**: 2026.06.26
- **用途**: 底层设计系统，直接对接前端变量

## 💻 开发者使用

所有 Token 都可以直接在项目中引用：

\`\`\`css
/* 引入 CSS 变量 */
@import 'path/to/tokens.css';

/* 使用示例 */
.button {
  background-color: var(--color-brand-primary);
  padding: var(--space-2);
  border-radius: 6px;
}
\`\`\`
